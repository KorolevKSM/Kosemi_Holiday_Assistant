# Kosemi Holiday Assistant

**Первый модуль ИИ-агента Kosemi.**  
Автоматизация поздравлений с праздниками и управления контактами.

## Проблема, которую решает проект
Я ленив, чтобы помнить о днях рождениях и праздниках, но хочу поддерживать связь с важными людьми. Этот инструмент берёт социальное взаимодействие под контроль: следит за датами, генерирует персонализированные поздравления и отправляет их за меня.

## Ключевые принципы
- **Полная автономность** – никаких платных облаков, все данные под моим контролем.
- **Постепенное усложнение** – от простых скриптов до продвинутого ИИ-ассистента.
- **Безопасность и приватность** – контакты не передаются третьим лицам.

## Текущий стек
- Python 3.11+
- SQLite (хранение контактов, праздников и шаблонов)
- python-telegram-bot (интеграция с Telegram)
- schedule (планировщик задач)
- tkinter / PyQt (будущий GUI)
- (в перспективе) локальное LLM‑ядро Kosemi (Ollama + Qwen 2.5)

## Дорожная карта (уровни развития)

| Уровень | Название | Описание |
|---------|----------|----------|
| 1 | Автоматизация и шаблоны | Консольный прототип с базой контактов и коллекцией шаблонов поздравлений. Псевдослучайный выбор шаблона, подстановка имени. Отправка готового текста через Telegram-бота (себе или контакту). |
| 2 | Умная база контактов | Добавление интересов, заметок и приоритетов для каждого человека. Скрипты, которые учитывают близость отношений при выборе стиля поздравления. |
| 3 | Графический интерфейс | Удобный GUI для добавления, редактирования и просмотра контактов. Встроенный календарь с ближайшими праздниками и историей отправленных поздравлений. |
| 4 | Мультиканальная отправка | Модули для автоматической отправки через Telegram, ВКонтакте, email. Интеллектуальный выбор канала связи на основе предпочтений контакта. |
| 5 | ИИ‑ядро Kosemi | Полная интеграция с локальным LLM‑ядром. Генерация уникальных поздравлений на основе данных о человеке, предпросмотр и редактирование в GUI перед отправкой. |

**Текущий статус:** [ ] Уровень 1 (в активной разработке)
